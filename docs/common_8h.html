バージョンの表現方法

  • バージョンは "V2.1" のように、"メジャーバージョン" + "." + "マイナーバージョン" で表現します。
  • 内部的な表現は、メジャー・マイナーバージョンを組み合わせた 21 のような整数型にします。
  • バージョン情報は互換性の確認や動作の安定性評価に利用されます。

プロパティ

本体から設定できる値を「プロパティ」として管理します。
内部の表記では 下記のプロパティと対応する ID を列挙型の 'PropertyID' で管理しています。
また、各プロパティのとりうる値の内容も列挙型で管理しています。

設定 PropetyID 設定値の列挙型
電源周波数 POW_FREQ Type_powFreq
ハムフィルタ HUM_FILTER Type_humFilter
ドリフトフィルタ DRIFT_FILTER なし
筋電フィルタ MUS_FILTER Type_musFilter
筋電フィルタ強度 MUS_ST Type_musSt
QRS検出モード QRS_MODE Type_qrsMode
QRS感度（手動） QRS_SENSI Type_qrsSensi
ECG感度 ECG_SENSI Type_ecgSensi
QRS感度（自動） AUTO_QRS_ST Type_autoQrs
アンテナ番号 ANTENNA
チャンネル番号 CHANNEL
