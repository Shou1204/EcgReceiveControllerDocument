◆ HandshakeProcesses()

int HandshakeProcesses ( )

ハンドシェイクプロセスを実行

戻り値
  • HANDSHAKE_SUCCESS: ハンドシェイク成功
  • HANDSHAKE_FAILED : ハンドシェイク失敗

  • HANDSHAKE_WORKING: ハンドシェイクが進行中

    ハンドシェイクプロセスの進行状況に応じて、次の動作を実行します：

  • フォーマットの生成および送信
  • 成功シグナルの確認
  • 必要に応じたリトライ
