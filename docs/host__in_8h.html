int UpdateData ( )

データ更新プロセス

戻り値
更新なし、更新あり（ハンドシェイクなし）, 更新なし（ハンドシェイクあり）を返す

ここでは現在値での重複を確認し、なければUART入力値を現在値に反映する

ハンドシェイクが必要かを判断するにとどめる また、いくつのプロパティに更新があったかを集計している
